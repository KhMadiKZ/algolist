Расширенный алгоритм Евклида

В то время как "обычный" алгоритм Евклида находит наибольший общий делитель двух чисел A и B, расширенный алгоритм Евклида находит помимо НОД также коэффициенты X и Y такие, что:

A X + B Y = GCD(A,B)

Реализация

int gcd (int a, int b, int & x, int & y)
{
	if (a == 0)
	{
		x = 0; y = 1;
		return b;
	}
	int x1, y1;
	int d = gcd (b%a, a, x1, y1);
	x = y1 - (b / a) * x1;
	y = x1;
	return d;
}

Асимптотика

Асимптотика работы, очевидно, такая же, как и у обычного алгоритма Евклида - O (log max(A,B)); более подробно см. "алгоритм Евклида".
